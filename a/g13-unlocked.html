HTC A510e G13 野火(Wild Fire)解锁教程(s-on解锁unlocked)，完美Recovery root权限

  2012-03-22     我的涂鸦

G13入手也有一段时间了，money的原因买的是水货，拿回来后发现确实如网友所述操作系统里面的鸡肋软件太多，什么twitter facebook youtube之类的，在我朝泱泱大国这等流氓网站的东西怎么能这么横行，所以果断应该把这些乱七八糟的全部kill掉，但是网上一搜，s-on的机器无法删除系统自带软件，杯具了。

那继续搜索有木有能解锁s-on的呢，有是有，但是需要制作金卡，而且HTC A510e G13 野火(Wild Fire)的这个版本压根就没有找到能够成功解锁的教程，只能搁置了。

又是一个阳光明媚的日子，再次找度娘扒拉一下试图看看有木有牛银可以把G13解锁的，没想到还真的找到了精细，发现官方自己出了解锁教程了，那官方解锁地址：http://www.htcdev.com/bootloader/ 不过英文不好的可能就杯具了，这里把大体步骤翻译一下，还没成功的同学先认证看完之后再动手哦，不然变砖头别赖我撒。

HTC A510E G13官方解锁教程：

1. 需要下载的软件：

SDK：http://dl.google.com/android/installer_r16-windows.exe
Java：http://java.com/
HTC Sync：http://www.htc.com/www/help/

2. 解锁步骤

2.1 登录HTC官方解锁网址 http://www.htcdev.com/bootloader/ （没登录的请先注册再登录，注意邮箱需要激活），然后在右侧选择要解锁的机型 HTC Wildfire S* ,然后点击Begin Unlock Bootloader（开始解锁boot权限），接着会弹出提示框让你确认，点击OK，然后再次弹出提示框，大体意思就是如果解锁了那么HTC的一些默认东西就不能用了，而且不再提供免费保修了，直接全部勾选这两项然后点击Proceed to Unlock Instructions进入下一步；

2.2 这个页面是要告诉我们若要解锁需要先安装下面的RUU，在网页最下方给出了对应机型版本的RUU，大概几十M，我们只需要选择对应版本刷上就好，都是英文的名字，不懂的可以去找在线翻译，这里大家可以选择其中的HTC EU就是欧版的，如何刷入版本大家可以自行百度（其实就是把刷机包zip文件改名为PG76IMG.zip放在SD卡上，然后同时按住开机和音量-出来之后就会自动更新），这个刷入的过程很多人可能会失败，我也是反复折腾了很久，如果出现错误，大家可以通过制作金卡的方式重新刷一个版本，如何做金卡大家可以去百度，如果这一步顺利那恭喜你成功一大半了；

2.3 这时跳转到http://www.htcdev.com/bootloader/unlock-instructions页面，这里面一共四个步骤，很简单：拔插电池；同时按住（特别注意是按住）按音量减和电源键；用音量键上下调整到Bootloader项并用电源键确认；数据线连接到电脑。然后点击“Proceed To Step 5”跳转到下一步；

2.4 找到你安装SDK的文件夹\android-sdk ，点击打开SDK Manager.exe ，然后稍等一段时间可以看到可更新的内容，我们选择“Android SDK Platform Tools and Android SDK Tools ”进行更新。更新完毕之后可以在 \android-sdk\platform-tools 目录下找到 adb.exe, AdbWinApi.dll 两个文件，在android-sdk\tools 找到 fastboot.exe 文件，把这三个文件放到C盘的android文件夹里面；

2.5 开始菜单——运行——输入CMD 命令行中输入

cd c:\Android 然后回车

fastboot oem get_identifier_token 然后回车

2.6 出现一大堆字符，我们右键点击标记，然后用鼠标选中从 <<<<identifier token start >>>> 开始（包括这一行）到end结束（包括end这一行），然后点击右键（此时已经复制到剪贴板中），然后粘贴到 http://www.htcdev.com/bootloader/unlock-instructions/page-3 网页下面的文本框中，然后点击下面的 Submit。

2.7 邮箱收到的邮件，点击里面的链接，下载附件放到C盘的android文件夹 。

2.8 命令行中继续运行
fastboot flash unlocktoken Unlock_code.bin 音量键选择确认或者推出升级，然后电源键确认。

2.9 系统已经恢复出厂设置了，再进入hboot可以看到最上面一行是“UNLOCKED”，解锁成功，音量键选择上面的YES，不然就白费了。

3.解锁成功之后接下来的事情就是刷root和刷recovery了

3.1 其实如果前面做的都顺利，这两步就不是很难了，首先下载以下文件

root包root.zip：http://s.dbank.com/e0rpsjm0su 密码 haixin

Recovery包PG76IMG.zip: http://s.dbank.com/e0ihwd6v6f 密码 haixin

3.2 先来刷recovery，步骤如下：

把3.1中下载好的 PG76IMG.zip放到SD卡根目录下，关机，卸掉电池，现在拿起手机中HTC G13，同时按住音量下键 和 手机开机键，进入HBoot界面,等待读卡后，在手机屏幕下方出现的两行红色字幕的选项，根据提示按音量上键，选择YES。

注：此步可能出现 version older 之类的版本太老的提示，那我们就打开压缩包中的android-info.txt 文件，使用记事本编辑其中的 mainver: 后面的数字，将其改为2.35.898（如果还是提示older那可以继续改大）

Recovery升级完毕后，会出现Recovery - OK的提示，然后再次根据提示，按电源键重新启动手机，成功后删除SD卡上的PG76IMG.zip，否则可能反复重刷。

3.2 永久root，最令人激动的时刻来了

① 将3.1中下载的root.zip放在内存卡上，关机，开机进入工程模式，音量键，选择recovery，进入主界面后，选第五项，从“内存卡更新”，进入后选第二项，从内存卡中查找到root，确认，再确认（英文的话就是yes和no，很多个no只有一个yes）。等待刷新即可，最后返回到首页，重启，完成。

② 开机看程序中会多一个超级管理员的安卓图标，那么恭喜你成功了，安装后会提示获取最高权限，获取后随便你删，建议尽量删第三方程序，我本人推荐直接重刷一个系统
