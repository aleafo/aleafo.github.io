Ecshop前台可选列表显示数量

  2011-06-27     PHP     ecshop

Ecshop前台可以通过点击链接选择每页显示的数量，这样让用户有更多的自主选择权力。默认的Ecshop只能通过后台的商店设置来修改每页显示的条数，本文通过更改程序文件实现Ecshop前台可选列表显示数量。实现步骤如下：

1. 修改search.php和category.php文件接收传递的page_size参数。

search.php大约第244行，

    $size       = !empty($_CFG[‘page_size’]) && intval($_CFG[‘page_size’]) > 0 ? intval($_CFG[‘page_size’]) : 10;

修改为：

    if (isset($_REQUEST[‘page_size’])){
    $size = intval($_REQUEST[‘page_size’]);
    setcookie(‘ecs_page_size’,$size,time()+3600);
}elseif(isset($_COOKIE[‘ecs_page_size’])){
    $size = intval($_COOKIE[‘ecs_page_size’]);
}else{
    $size = isset($_CFG[‘page_size’])  && intval($_CFG[‘page_size’]) > 0 ? intval($_CFG[‘page_size’]) : 10;
}

在category.php中做同样的修改。

2. 在/themes/default/category.dwt以及search.dwt中使用下面的代码调用(其中的category.php在search.dwt中应改为search.php)：

Items per page: <a href=”category.php?category={$category}&display={$pager.display}&brand={$brand_id}&price_min={$price_min}&price_max={$price_max}&filter_attr={$filter_attr}&page={$pager.page}&order=<!– {if $pager.sort == ‘last_update’ && $pager.order == ‘DESC’} –>ASC<!– {else} –>DESC<!– {/if} –>&page_size=20#goods_list”>20</a>

其中的20可以修改为其他任意你想要显示的条数。
