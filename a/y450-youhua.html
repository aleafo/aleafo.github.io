联想(lenovo)Y450重装（新装）系统手动优化步骤

  2011-04-08     我的涂鸦

1. 去掉桌面右键菜单中的 nview和 Nvidia选项

注册表中找到HKEY_CLASSES_ROOT\Directory\Background\shellex\ContextMenuHandlers 删除对应项即可

2. 去掉新建菜单中的无用的杂项

注册表中找到 HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Discardable\PostSetup\ShellNew 删除对应项目即可

这个注册表可以解决 方法:在开始–运行 中输入 regedit 打开注册表编辑器 例如:文本文档 .TXT 在HKEY-CLASSES-ROOT-TXT-SHELLNEW 如果重命名该子键为 HKEY-CLASSES-ROOT-.TXT-SHELLOLD

注意 要知道新建项的后缀 文本文档的 后缀是.TXT 就可以将SHELLNEW 改为SHELLOLD 即可(其中公文包是 .bfc)

3. 去掉不需要的开机启动项（包括右下角的蓝牙图标等）

运行 msconfig，在启动里面将除了ctfmon.exe（这个是输入法，禁用之后就不能打字了）之外的所有启动项目勾掉。然后再服务里（services.msc)禁用NVIDIA Display Driver Service服务，并将启动方式改为手动,否则关于显卡的开机启动项会在重启后再次变为开机启动。

4. 移除桌面上的 我的bluetooth位置 图标

只需右击桌面，属性，桌面，自定义桌面，运行清理向导，将蓝牙的图标勾掉，下一步即可。

5. 禁用不需要的服务，加快开机速度，节省内存开销

关闭下面的服务即可（开始——运行——services.msc来打开服务管理器）：
NetMeeting Remote Desktop Sharing
Universal Plug and Play Device Host
Messenger
Terminal Services
Remote Registry
Fast User Switching Compatibility
Telnet
Performance Logs And Alerts
Remote Desktop Help Session Manager （远程协助）
TCP/IP NetBIOS Helper （打印机共享）

6. 减少不必要的磁盘开销

在系统正常运行的情况下，就三个原因会导致磁盘空间减少

虚拟内存　１－２倍物理内存的大小
磁盘还原 15%的本磁盘的大小
启用休眠 500-700MB

7. 在组策略中关闭U盘的自动播放，防止autorun等病毒

组策略的打开方式为：开始——运行——gpedit.msc
