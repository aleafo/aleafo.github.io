腾讯、新浪微博分享链接格式

  2011-06-08     HTML

最近微博很火，所以我们也要跟上脚步。 看到很多分享都是来自 XXX，这样的应用都是开发者使用了appkey实现的。

那本文就告诉大家关于新浪的分享链接的使用方法。本文以新浪微博为例来说明：

1. 到新浪开发平台申请一个appkey：http://open.weibo.com

2. 在新浪微博注册成为开发者，并填写要开发的应用信息确认后会显示一个appkey 和 secretkey给你；

3. 接下来就可使用上面的appkey开发你的应用了。新浪微博分享链接格式如下：

switch($share){
        case “sina”:
            $link = ‘http://v.t.sina.com.cn/share/share.php?appkey=4052864631&url='.urlencode($post_link).'&title='.urlencode($post_title) .’&source=&sourceUrl=&content=utf-8&pic=’.urlencode($pic);
            break;
        case “qq”:
            $link = ‘http://v.t.qq.com/share/share.php?appkey=90ef6a523cb94c28ace7efcc4cae26c8&url='.urlencode($post_link).'&title='.urlencode($post_title) .’&site=&pic=’.urlencode($pic);
            break;
        default:
            break;
    }

上文的示例代码为PHP的示例代码，大家可以修改响应的参数从而实现分享到对应的微博即可。
示例：

http://v.t.sina.com.cn/share/share.php?appkey=4052864631&url=http://www.lovelai.com&title=来爱吧，青岛婚恋交友网站&source=&sourceUrl=&content=来爱吧，青岛婚恋交友网站&pic=http://www.lovelai.com/images/e-logo.gif

这段代码即是将lovelai.com这个网站分享到新浪微博的方式。你们可以自行修改其中的代码分享到其他网站。
