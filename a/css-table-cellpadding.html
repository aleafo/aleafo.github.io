CSS控制表格的cellpadding/cellspacing

  2011-08-06     HTML     CSS

今天翻开百度知道，看到有个小盆友问表格的cellpadding 和 cellspacing如何使用css设置为0。看了之后，自觉惭愧，搞了这么久的网页设计一直以来的方法碰到表格都直接在table里写入，也从来没有考虑过这个问题。于是谷歌了一下，发现已经有答案了。贴个地址来。用css控制cellspacing、cellpadding。文章简短明了。

table{border:0;margin:0;border-collapse:collapse;}
table td{padding:0;}

第一行控制cellspacing，关键是“border-collapse:collapse;”这一句；第二行控制cellpadding，很少人知道控制td吧？

去w3school看了下关于border-collapse的解释：border-collapse 属性设置表格的边框是否被合并为一个单一的边框，还是象在标准的 HTML 中那样分开显示。所有主流浏览器都支持 border-collapse 属性。

separate    默认值。边框会被分开。不会忽略 border-spacing 和 empty-cells 属性。
collapse    如果可能，边框会合并为一个单一的边框。会忽略 border-spacing 和 empty-cells 属性。

现在再来看table的这个属性设置就比较明白了。以后还是使用这种方式设置表格的cellspacing和cellpadding比较好，因为这样更符合w3c的标准。
