按键精灵 vbs 获取网页源码 xp系统被拒绝

  2015-03-10     Javascript

如下面的代码所示，获取新浪博客某个指定网页的源码


verurl = “http://blog.sina.com.cn/s/blog_9ea1db7b0101o7ch.html?“ & now()
Set Http = CreateObject(“Microsoft.XMLHTTP”)
Http.open “get”, verurl, False
Http.send
Delay 50
vbody = Http.responsebody
TracePrint vbody

　　结果在某个XP系统下会出现“拒绝访问”的提示，更换为百度等网址后不再出现拒绝访问的提示，而是返回空白，但是用IE浏览器打开该网页可以正常显示。更换了一台电脑测试XP系统和WIN7系统均可以正常执行并返回结果。问题很诡异，暂时存疑，未解决。
