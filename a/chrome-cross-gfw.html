Chrome/谷歌浏览器 + Auto Proxy 5分钟学会全自动翻·墙大法

  2011-05-31     我的涂鸦

话说自从有了功夫网（Great FireWall），从一定程度上限制了很多人上“不良网站”。但是只要功夫深，大墙不如针。 使用Firefox的童鞋们可以很容易的下载Auto Proxy 来翻墙，连列表都给你加载好了，只要下载你需要的软件就可以了。本文介绍个人追崇的Google Chrome（谷歌浏览器）的设置方法。也就是使用Proxy Switchy! + MyEnTunnel + Chrome浏览器 实现有墙似无墙之翻·墙大法。

步骤如下（本人亲测可用，测试日期2011.05.31）：

1. 需要的软件：

Google Chrome 浏览器

MyEnTunnel （网上很多地址被墙，这里下载后请修改扩展名为.RAR解压即可）

Proxy Switchy! 大家可以点击这里从谷歌下载 ,如果页面无法访问了（可能会被墙）那就点这里 - Proxy Switchy! 从本站下载后修改扩展名为RAR，然后将其放到谷歌浏览器插件目录下。

2. 注册SSH帐号

大家可以通过下面的地址申请一个SSH帐号，免费的可以使用24小时，激活邮箱后可以使用24*2小时，过期了自己再去重新申请，对于一般用户来说免费的就够用了，付费的也不贵，对于经常fq的用户来说购买一个付费账户也是一个不错的选择。

http://www.sshcenter.info/?ref=69369

3. 启动MyEnTunnel

启动后填入各项内容 如下：

SSH服务器： ssh.sshcenter.info

SSH端口： 22

用户名、密码短语为你在第2步中注册的帐号和密码

后面的参数建议不要修改，如果你知道他们是什么意思那你就可以随便设置了。设置完以上内容后点击连接，当右下角图标变为蓝色的时候代表连接成功。

4. 设置Proxy Switchy!

接下来启动谷歌浏览器，然后点击右上角的Proxy Switchy地球形状的图标——Options 进入设置（或者扳手按钮——设置——扩展程序——找到Proxy Switchy——点击选项）。

在Proxy Profiles选项卡中新建一个Profile 名字随便起，这里我们命名为SSH-D，选中 Manual Configuration 然后在SOCKS Host中输入本机IP 127.0.0.1 端口填写7070（如果上面你修改了MyEnTunnel的端口（不是SSH端口）那这里同样修改为相同的即可。然后选择Socks v5，最后点Save（保存）

在Switch Rules选项卡中勾选Online Rule List，然后在Rule list URL中输入地址：https://switchy-list.googlecode.com/svn/trunk/SwitchyRules.txt，后面的Proxy Profile 选择前面命名的SSH-D，不要勾选AutoProxy Compatible List，否则会出错。然后点击Save保存设置。

最后点击右上角的Proxy switchy 图标，选择Auto Switch mode。 你就学会了有墙似无墙之翻【河蟹】墙大法。

需要注意的

这里我们要注意的是，本文并没有引用Firefox中使用的Auto Proxy代理的在线列表地址，而是使用了单独为Proxy Switchy维护的另外一个列表，这是本文的重点。最初使用Auto Proxy在线列表需要勾选AutoProxy Compatible List，但是测试多次重启浏览器甚至重启电脑都没有成功，原因不明。
