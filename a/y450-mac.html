实施步骤如下，首先给大家一个 新手安装指南：一步一步在Windows安装苹果雪豹系统 或者：新手windows下安装苹果教程 - 最详细，最全面，最值得看的教程！，仔细通读上面的教程，然后再进行安装。多花点时间准备上，比你到后面碰到问题再解决要好的多，工欲善其事必先利其器。

下面说说自己的安装注意事项和安装驱动（我大部分时间都花在了安装驱动上面了）的注意事项，请先把这部分看完再开始安装。

1. 安装注意事项：

安装Mac之前，记得bootthink驱动Kext文件，也就是C盘的 /Darwin/System/LibrarySL/Extensions 里面的文件只保留fakesmc.kext和OpenHaltRestart.kext这两个，其他到全部移动到别的地方备用，同时把/Darwin/dsdt.aml也移动到别处备用。如果不这么做，你的苹果系统安装完成后很可能五国（出现五国语言，只能断电重启）等，那就只能进win系统删除kext和dsdt.aml了；

安装Mac系统的时候，注意在前面的安装向导部分，记得把打印机的选项勾去掉，否则安装会很慢、时间很长；

安装进行的过程中，一定记得每隔几分钟动一下鼠标或者拍一下键盘，否则联想Y450可能会进入硬盘休眠状态，那安装就费费了；

如果安装到最后只剩一点，然后一直不动，可以强制断电，可能已经安装成功了，重启试试看。

安装完成后，先去官方网站下载Mac OS X v10.6.6的组合更新升级包（可以在win系统下访问网站下载），不要直接在线升级，那会直接升级到10.6.8或更高，重启后直接无法进入系统，删除kext都不行，切记。

2. 安装驱动及其注意事项

2.1 boothink及其驱动下载

bootthink是国产的用于安装多系统的引导工具，直接解压后放在C盘根目录，然后运行其中的install.bat(win7下必须以管理员身份运行)，即可安装成功。

关于DSDT，DSDT.aml文件是用来描述苹果系统的电脑硬件信息的，mac可以根据这里的内容来正确的识别电脑的硬件信息。很多牛人也都根据不同的硬件制作了不同的dsdt在论坛里面可以很容易的搜素到。

打包好的bootthink（含远景zwt提供的这个2010年12月11更新的dsdt可以调节屏幕亮度的以及Y450 TSI GT240M for Mac OS 10.6.6完美驱动下载）下载地址

2.2 Y450 mac OS 10.6网卡驱动

Y450 mac OS 10.6的网卡是BCM5754，没有这个型号的标准驱动。驱动能直接使用的kext文件非常难找，所以这里建议大家使用修改过的AppleBCM5701Ethernet.kext （已经包含在2.1中）直接复制到 SLE下（mac系统的 /System/Library/Extensions/ 中文显示应为 系统->资源库->Extensions），然后使用命令修复权限，重启即可。修复权限的步骤如下：
应用程序——实用工具——终端，输入

sudo -s 回车 （显示Password:即输入密码，你输入的时候是看不到的，输入完了回车即可）
chown -R root:wheel /system/Library/Extensions/AppleBCM5701Ethernet.kext
chmod -R 755 /System/Library/Extensions/AppleBCM5701Ethernet.kext
exit
reboot

上面第2行赋予使用者为root，第3行设置读写权限为755,第4行exit为退出，最后是重启电脑。

Y450 Mac OS 无线网卡驱动，因为5100暂时无解，所以如果非常需要这个功能，那建议先去换个网卡把原配的英特尔5100无线网卡换成Atheros AR9280，此卡在雪豹下自动驱动。

2.3 Y450 Mac OS声卡驱动

Y450 mac OS 的声卡驱动方式有两种，第一是使用VoodooHDA.kext 这个是其中的一种方法，但是我测试过之后总是有问题，所以不推荐；
第二种驱动动方法是使用两个文件 AppleHDA.kext 和 LegencyAppleHDA.kext，将这两个文件直接复制到SLE下，修复权限然后重启即可（方法同2.2网卡驱动）

2.4 其他驱动

其他驱动直接放在/Darwin/System/LibrarySL/Extensions下即可，这里直接使用本文中2.1 的bootthink里面自带的驱动kext文件即可全部正常驱动了。

3 其他说明及注意事项

3.1 Y450 Mac OS Fn+上下箭头调节亮度问题：其实这个DSDT文件设置的并不是使用Fn+上下箭头，而是Fn+F9和Fn+F11来调节亮度的。

3.2 Y450 Mac OS 设置开机亮度问题，这个问题可以参考这里 http://bbs.pcbeta.com/viewthread-830707-1-1.html

3.3 Y450 Mac OS 每隔一段时间光驱就会响一次（光驱总响，吱嘎声），这是硬盘的节电功能导致的，打开偏好设置，在节能器里面将“如果可能，使硬盘进入睡眠”的勾去掉即可

3.4 Mac OS文件路径怎么找，在Finder中上面的工具栏按钮空白区域右键——自定义工具栏——将路径拖入即可。
